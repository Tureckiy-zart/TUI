All exports are available from the main entry point unless otherwise specified.  ---  ## Canonical Context  **Source of Truth:** This document reflects the actual public API exported from `src/index.ts`.  **Foundation Components:** See [FOUNDATION_LOCK.md](../architecture/FOUNDATION_LOCK.md) for Foundation component lock status and immutability rules.  **Extension Components:** See [EXTENSION_STATE.md](../architecture/EXTENSION_STATE.md) for complete Extension layer component classification (ALLOWED/RESTRICTED/INTERNAL).  ---  ## Entry Points  The library provides multiple entry points for tree-shaking and selective imports:  | Entry Point | Purpose                     | Example                                                    | | ----------- | --------------------------- | ---------------------------------------------------------- | | `.`         | Main entry (all components) | `import { Button } from '@tenerife.music/ui'`              | | `./styles`  | CSS styles only             | `import '@tenerife.music/ui/styles'`                       | | `./preset`  | Tailwind preset             | `import preset from '@tenerife.music/ui/preset'`           | | `./tokens`  | Design tokens only          | `import { primaryColors, semanticSpacing } from '@tenerife.music/ui/tokens'`       | | `./theme`   | Theme system only           | `import { ThemeProvider } from '@tenerife.music/ui/theme'` |  ---  ## Component Classification  See [EXTENSION_STATE.md](../architecture/EXTENSION_STATE.md) for complete component classification rules (ALLOWED/RESTRICTED/INTERNAL).
